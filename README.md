# 🌐 Front-End Web Projects

A collection of modern front-end web projects built using HTML5, CSS3, Bootstrap, JavaScript, and jQuery. This repository showcases responsive landing pages, business websites, and UI-focused designs created to strengthen front-end development skills and modern web design practices.

## 🚀 Projects Included

### 🏢 AXIT Landing Page

A modern landing page featuring:

* Hero Section
* Pricing Plans
* Testimonials
* Contact Form
* Social Media Integration
* Responsive Layout

### 🐾 Paws n' Play

A pet store landing page featuring:

* Product Showcase
* Pet Services
* Interactive Hover Effects
* Modern UI Components
* Responsive Design

### 💼 Trade-X Business Website

A corporate website featuring:

* Multi-Page Design
* Bootstrap Components
* Contact Page
* Form Validation
* Animated Counters
* Testimonials Carousel
* jQuery & Owl Carousel Integration

### 🖥️ Nexcent Landing Page

A business-focused landing page featuring:

* Community Management Section
* Statistics Dashboard
* Customer Testimonials
* Blog Cards
* Newsletter Subscription
* Call-To-Action Sections
