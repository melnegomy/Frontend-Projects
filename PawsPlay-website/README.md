# 🐾 Paws n' Play Pet Store Website

A modern pet store landing page built using HTML5 and CSS3. The website provides an attractive user interface for pet lovers, featuring product showcases, pet services, and a visually engaging design focused on user experience.

## 🚀 Features

* 🏠 Responsive Navigation Bar
* 🎨 Hero Banner Section
* 🛍️ Featured Products Showcase
* 🔍 Product Search Bar
* 🐶 Pet Categories Section
* ✂️ Services Section

  * Grooming
  * Boarding
  * Veterinary Care
  * Training
  * Adoption
* ✨ Hover Overlay Effects
* 📱 Responsive Layout
* 🎯 Modern UI Design
