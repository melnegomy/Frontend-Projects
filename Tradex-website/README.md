# 💼 Trade-X Business Website

A modern multi-page business website built using HTML5, CSS3, Bootstrap 5, JavaScript, jQuery, and Owl Carousel. The project showcases a professional corporate website with interactive UI components, responsive layouts, animated counters, testimonials, and contact forms.

## 🚀 Features

* 🏠 Responsive Home Page
* 📞 Contact Page
* 🎠 Hero Carousel Slider
* 📊 Animated Statistics Counter
* 👥 Testimonials Carousel
* 💼 Services Section
* ℹ️ About Us Section
* 📬 Newsletter Subscription
* 🏢 Client Showcase Section
* 📋 Contact Form Validation
* 🎨 Interactive Hover Effects
* 📱 Fully Responsive Design
