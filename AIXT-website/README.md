# 🎨 AXIT Landing Page

A modern and responsive landing page built using HTML and CSS. The project recreates a professional marketing website layout with multiple sections including a hero banner, pricing plans, testimonials, social media integration, and a contact form.

## 🚀 Features

* 🧭 Navigation Bar
* 🖼️ Hero Banner Section
* 📝 Free Trial Form
* 🌐 Social Media Section
* 📑 Tabs Section
* 📋 Features / Sub List Section
* 💰 Pricing Plans
* 💬 Customer Testimonials
* 📞 Contact Form
* 🔗 Footer with Social Icons
